Stage 2 Preview / Expected Review Format

Section feedback

If scope is approved, Stage 2 will update this same review page with a complete rendered working packet. The Markdown working packet will be supplemental; the primary review surface will be structured HTML.

  1. Research Scope Approved: approved mode, sources, output paths, caveats, and any user edits.
  2. Executive Findings: only after research, with confidence labels and no unsupported recommendations.
  3. Marketplace Side Preflight: buyer, user, and respondent side coverage, including explicit exclusions.
  4. Broad Research Log: query strategy table, source categories, observations, and gaps.
  5. ICP Candidate Review: 2-5 candidates, pain evidence, accessibility, value potential, and WTP signal strength.
  6. Framework Selection: defaults and optional frameworks matched to the approved mode.
  7. Evidence Matrix: claims, source/path, inference, confidence, assumptions, and decision impact.
  8. Assumptions / Confidence Register: what is evidence-backed, provisional, or changeable.
  9. Proposed File Changes: exact Stage 3 canonical artifacts and any task updates after approval.
  10. User Format Preferences Gate: layout, tables, grouping, visuals, labels, and evidence density.
  11. Final Artifact Approval Gate: required before canonical files are created or updated.
